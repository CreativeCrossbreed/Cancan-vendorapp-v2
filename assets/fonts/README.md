# Agrandir Font Files

Please place your Agrandir font files in this directory.

The app is configured to use:
- **Agrandir-Regular.otf** or **Agrandir-Regular.ttf** (for regular weight)
- **Agrandir-Bold.otf** or **Agrandir-Bold.ttf** (for bold weight)

If your font files have different names, please update the `pubspec.yaml` file accordingly.

Common naming patterns:
- Agrandir-Regular.otf / Agrandir-Bold.otf
- Agrandir-Regular.ttf / Agrandir-Bold.ttf
- Agrandir_Regular.otf / Agrandir_Bold.otf
- AgrandirRegular.otf / AgrandirBold.otf
